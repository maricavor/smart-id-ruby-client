# smart_id

Ruby client gem for integrating Smart-ID into Ruby applications.

## Installation

Install the gem and add to the application's Gemfile:

```bash
bundle add smart_id
```

Or install it directly:

```bash
gem install smart_id
```

## Usage

```ruby
require "smart_id"
```

## Development

After checking out the repo, run:

```bash
bin/setup
bundle exec rake
```
